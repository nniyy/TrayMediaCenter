# Media Tray Center
<img width="100%" src="./docs/preview.gif"/>

## Features
- Left click: Play/Pause
- Shift + Left click: Previous track
- Double click: Next track
- Context menu: Play/Pause, Next, Previous
- Shows track/artist or source in tooltip
- Optional autostart

## Requirements
- Windows 10/11
